# Wasteless Logo

Place the Wasteless logo file here as `logo.png`.

The logo will be displayed in:
- Sidebar (full width)
- Main page header (120px width)

## Required

Save the logo image as: `logo.png`

Supported formats: PNG (recommended), JPG, SVG

## Logo Specifications

- **Recommended size**: 600x200px or similar aspect ratio
- **Format**: PNG with transparent background
- **Color**: Green "less" with gray "waste"
- **Aspect ratio**: ~3:1 (horizontal)
